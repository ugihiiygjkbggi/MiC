# 🚀 C# 與程式開發 Markdown 語法全攻略

## 1. 標題 (Headings)
使用 # 開頭，符號與文字間要有一個空格。
#代表一級標題
##二級標題
###三級標題
如下
# 一級標題
## 二級標題
### 三級標題

---

## 2. 程式碼呈現 (Code)

### 行內程式碼 (Inline Code)
語法是兩個\`程式碼\`符號
用於文字間：`var myData = 10;`
### 程式碼區塊 (Code Blocks)
語法:\```csharp

```csharp
// 這裡可以寫完整的程式邏輯
public void HelloWorld() 
{
    Console.WriteLine("Hello Markdown!");
}
```

---

## 3. 條列清單 (Lists)
語法:\*、1.，記得加空白建在\*和文字間
* **無序清單** (使用 * 或 -)
  * 第一層項目
  * 第二層項目
* **有序清單** (使用 1. 2. 3.)
  1. 第一步驟
  2. 第二步驟

## 6. 數學公式 (LaTeX)

行內公式：$C_{max} = \sum p_j$

獨立區塊公式：
$$
C_{max} = \max_{j=1 \dots n} \{ C_{mj} \}
$$
